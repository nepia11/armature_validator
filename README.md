## 概要

アーマチュア A,B の構造を検証する機能を実装する

## install
download

## usage

1. アーマチュアを複数選択する
2. 基準にしたいアーマチュアをアクティブにする
3. 右クリックメニューから`check hierarchy`または`show hierarchy`を実行する
4. 結果が情報エリアに出力される

### check hierarchy

アーマチュアの A,B の比較をする

表示される結果の読み方

```
invalid_parent: A と B で親が違うボーンの名前
only_A: A にのみ存在するボーン
only_B: B にのみ存在するボーン
```
